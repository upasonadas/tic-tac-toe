# Tic Tac Toe Game 

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
This project helps in understanding DOM manipulation, event handling, and basic game logic.

# Features
- Two-player game (X and O)
- Interactive user interface
- Winner detection
- Draw condition handling
- Reset / Restart game option

# Technologies Used
- HTML
- CSS
- JavaScript
